
# Ruby Enumerable methods

> This project demonstrates the recreation of some of the methods that are part of the Ruby Enumerable module.

## Methods Rebuilt

- .each
- .each_with_index
- .select
- .all?
- .any?
- .none?
- .count
- .map
- .inject

## Built With

- Ruby
- Rubocop
